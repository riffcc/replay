# replay

Replay is a radically simple attempt at software orchestration - to build, manage, maintain and improve software automatically.

Replay has a few interesting properties:
* Explicitly non-linear execution
  It does not assume everything is a pipeline. Tasks can loop, check, and reintroduce.

* Event-driven
  Everything is based around events.
